# 📊 CGPA & GPA Calculator Web App

This project is a **web-based CGPA & GPA calculator** with student registration and OracleDB integration. Users can enter their academic details, calculate GPA/CGPA, and store the results in a database.

---

## 🚀 Features

- 🎓 GPA & CGPA Calculation (Dynamic input)
- 📝 Student Registration with:
  - Name
  - Register Number
  - Department
  - Admission Type (Regular, Lateral, Self-Support)
- 📦 Data stored in Oracle Database
- 📊 Auto-sort into department-wise and admission-wise tables
- 🧠 Trigger to maintain admission count
- 🧹 Option to clear all student data
- 🌐 Full frontend-backend integration

---

## 🛠️ Tech Stack

| Frontend  | Backend        | Database |
|-----------|----------------|----------|
